# Devin's Emergency Roofers LLC Website

Static multi-page website for a local roofing contractor in Provo, Utah.

## Pages
- `index.html` — Home
- `services.html` — Services overview
- `emergency-roof-repair.html`
- `roof-replacement.html`
- `storm-damage-insurance-claims.html`
- `gutters.html`
- `about.html`
- `reviews.html`
- `contact.html`
- `styles.css` — shared styling

## Notes
- Mobile-first, lightweight static site
- Click-to-call phone links throughout
- Sticky mobile call button
- Local SEO-focused titles and descriptions
- Review quotes use only approved 5-star source material
- Contact form is front-end only and should be wired to Formspree, Netlify Forms, Basin, or a backend endpoint before production
- Replace placeholder map/embed and imagery as needed

## Deploy
Any static hosting works:
- Netlify
- Vercel static
- Cloudflare Pages
- GitHub Pages
- S3/CloudFront
